# NGO / Charity Website Template

### React + Tailwind CSS

Thank you for purchasing this NGO & Charity Website Template.
This template is built with **React, Tailwind CSS, and Vite** and is designed to help you quickly create a modern website for NGOs, nonprofits, charity organizations, or fundraising campaigns.

---

# 📦 What’s Included

After downloading the template, you will get:

* Complete React source code
* Tailwind CSS styling
* Reusable components
* Responsive layout
* Clean and organized folder structure

Example structure:

```
project-folder
│
├── public
├── src
│   ├── assets
│   ├── components
│   ├── pages
│   ├── App.jsx
│   ├── main.jsx
│

```

# ⚙️ Requirements

Before running this template, make sure you have installed:

* Node.js (v16 or newer recommended)
* npm or yarn

Check versions:

```
node -v
npm -v
```

---

# 🚀 Installation Guide

Follow these steps to run the template.

### 1. Extract the Files

Download and extract the template ZIP file.

```
ngo-template.zip
```

Open the extracted folder in **VS Code** or any code editor.

---

### 2. Install Dependencies

Open the terminal in the project folder and run:

```
npm install
```

This will install all required packages.

---

### 3. Start Development Server

Run the project with:

```
npm run dev
```

You will see a link like:

```
http://localhost:5173
```

Open this link in your browser.

# 🧑‍💻 How to Customize

### Change Website Name

Edit:

```
src/components/Navbar.jsx
```

Example:

```
HelpFoundation
```

Replace with your organization name.

---

### Change Images

Images are stored in:

```
src/assets
```

You can replace them with your own images.

Example:

```
hero.jpg
volunteer.jpg
campaign.jpg
```

---

### Edit Text Content

Most content is inside components:

```
src/components
```

Example:

* Hero section → `Hero.jsx`
* Campaign cards → `Campaigns.jsx`
* Testimonials → `Testimonials.jsx`
* Contact section → `Contact.jsx`

---

# 🎨 Change Colors
